Todo Application

Description of application

Documentation

  • POST /todos Creates a new todo
    Expected request message body: 
    {
  "todo": {
    "description": "Buy groceries"
  }
}
  • GET /todos Returns list of todos
    Response message body: 
    { "todos": 
  [ 
    {
      "id": "9b60db4c-2042-4516-9a37-fb916465de7c",
      "done": false,
      "description": "Buy cat food"
    }
  ]
}
